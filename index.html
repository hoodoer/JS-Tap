JS-Tap Guide
There are a few settings in the JS-Tap payload that need to be set prior
to your engagement. The primary JavaScript payload is contained in telemlib.js.

JS-Tap has two modes of operation, Trap mode and Implant mode.
Trap mode is what you use as an XSS payload, and it uses
an iFrame trap as a method of persisting the execution of the payload.

Implant mode does not use this persistence method, and assumes the
payload has been directly added to the JavaScript files of the application
by the red team.

In the initGlobals() function you need to set the start page of the iFrame trap
and the jsTapServer hostname/port.
