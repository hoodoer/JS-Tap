Mimic Form Submission

Instructions:
This modal allows you to configure a custom payload that will mimic this form submission. You'll be able to modify the parameters in the custom payload.

First we need to check if the submission uses a CSRF token. We'll need to build a more complicated payload if the request uses such a token. These tokens are a random value and the parameter name is typically related to CSRF, XSRF, or Nonce. If you see a parameter like that, enter the parameter name and value on the right and hit search. JS-Tap will search it's previously captured HTML looking for where this secret value comes from and build a payload to find and parse out this value. It will be needed for the part of the payload that submits the action you're mimicking.

If there is no random CSRF value apparent in the form submission hit the "Next" button to build the payload.

Mimic Network API Call

Instructions:
This modal allows you to automatically build a custom payload that will mimic this network call. You'll be able to modify the parameters in the custom payload.

First we need to check if the API call uses an authorization header or token. We'll need to build a more complicated payload if the request uses such an auth token. These tokens are typically sent in an Authorization header. If you see a header like that, enter the header name and value on the right and hit search. These are often in the format:
Authorization: Bearer SECRETTOKEN

Do not include the Bearer string for the search, use Authorization for the token name and SECRETTOKEN as the value for the search. The payload generator will automatically add the Bearer string if needed.
JS-Tap will search it's previously captured storage looking for where this secret value is stored so it can build a payload to find this auth token. It will be needed for the part of the payload that submits the action you're mimicking.

If there is no authorization token perhaps the API relies on cookies you can't see, and you should be able to hit the "Next" button to build the payload.
